The truth of the next proposition may be clear, but it is included to illustrate the process of proving one set is a subset of another set. In this proposition, the set \(S\) is the set of all integers that are a multiple of 6. So when we “choose” an element from \(S\text{,}\) we are not selecting a specific element in \(S\) (such as 12 or 24), but rather we are selecting an arbitrary element of \(S\) and so the only thing we can assume is that the element is a multiple of 6.
