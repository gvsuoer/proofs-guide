We will prove the contrapositive of this statement, which is

For each positive real number \(x\text{,}\) if \(\sqrt{x}\) is rational, then \(x\) is rational.
So we assume that \(x\) is a rational number and that \(\sqrt{x}\) is rational, and will prove that \(x\) is rational. Since \(\sqrt{x}\) is rational, there exist positive integers \(m\) and \(n\) such that \(\sqrt{x} = \frac{m}{n}\text{,}\) then \(x = \frac{m^2}{n^2}\text{.}\) Since \(m\) and \(n\) are positive integers, \(m^2\) and \(n^2\) are positive integers and we can conclude that \(x\) is a rational number. This proves the contrapositive of the statement and so we have proved that for each positive real number \(x\text{,}\) if \(x\) is irrational, then \(\sqrt{x}\) is irrational.
