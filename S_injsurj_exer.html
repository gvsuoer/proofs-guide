Section 7.3 Practice Problems for Chapter 7

Exercises

1.

Let \(R^+ = \{ y \in \R | y \gt 0 \}\text{.}\) Define

\(f: \R \to \R\) by \(f(x) = e^{-x}\text{,}\) for each \(x \in \R\text{,}\) and

\(g: \R \to \R ^+\) by \(g(x) = e^{-x}\text{,}\) for each \(x \in \R\)

Determine if each of these functions is an injection or a surjection. Justify your conclusions

Note: Before writing proofs, it might be helpful to draw the graph of \(y = e^{-x}\text{.}\) A reasonable graph can be obtained using \(-3 \leq x \leq 3\) and \(-2 \leq y \leq 10\text{.}\) Please keep in mind that the graph does not prove any conclusion, but may help us arrive at the correct conclusions, which will still need proof.

Solution.

The function \(f\) is an injection but not a surjection. To see that it is an injection, let \(a, b \in \R\) and assume that \(f (a) = f (b)\text{.}\) This implies that \(e^{-a} = e^{-b} \text{.}\) Now use the natural logarithm function to prove that \(a = b\text{.}\) Since \(e^{-x} \gt 0\) for each real number \(x\text{,}\) there is no \(x \in \R\) such that \(f (x) = -1\text{.}\) So \(f\) is not a surjection.

The function \(g\) is an injection and is a surjection. The proof that \(g\) is an injection is basically the same as the proof that \(f\) is an injection. To prove that \(g\) is a surjection, let \(b \in R+\text{.}\) To construct the real number \(a\) such that \(g(a) = b\text{,}\) solve the equation \(e^{-a} = b\) for \(a\text{.}\) The solution is \(a = - \ln b\text{.}\) It can then be verified that \(g(a) = b\text{.}\)

2.

For each of the following functions, determine if the function is an injection or a surjection (or both, and hence, a bijection). Justify all conclusions.

(a)

\(F: \ \R \to \R\) defined by \(F (x) = 5x + 3\text{,}\) for all \(x \in \R\text{.}\)

Solution.

Let \(F: \ \R \to \R\) be defined by \(F (x) = 5x + 3\) for all \(x 2 R\text{.}\) Let \(x1, x2 \in \R\) and assume that \(F (x_1) = F (x_2) \text{.}\) Then,

\begin{align*} 5x_1 + 3 \amp = 5x_2 + 3\\ 5x_1 \amp = 5_x2\\ x_1 \amp = x_2 \end{align*}

Hence, \(F\) is an injection. Now, let \(y \in \R\text{.}\) Then, \(\frac{y - 3}{5} \in \R\) and

\begin{align*} F \left( \frac{y - 3}{5} \right) \amp = 5 \left( \frac{y - 3}{5} \right) + 3\\ \amp = ( y - 3) + 3\\ \amp = y\text{.} \end{align*}

Thus, \(F\) is a surjection and hence \(F\) is a bijection.

(b)

\(G: \Z \to \Z\) defined by \(G (x) = 5x + 3\text{,}\) for all \(x \in \Z\text{.}\)

Solution.

The proof that \(G\) is an injection is similar to the proof in Task 7.3.2.a that \(F\) is an injection. Now, for each \(x \in \Z\text{,}\) \(\modulo{5x + 3}{3}{5}\text{,}\) and hence \(\modulo{G(x)}{3}{5}\text{.}\) This means that there is no integer \(x\) such that \(G(x) = 0\text{.}\) Therefore, \(G\) is not a surjection.

(c)

\(f: ( \R - \{ 4 \}) \to \R )\) defined by \(f(x) = \frac{3x}{x - 4}\text{,}\) for all \(x \in ( \R - \{ 4 \} )\text{.}\)

Solution.

Let \(a, b \in \R - \{ 4 \}\) and assume that \(f(a) = f(b)\text{.}\) Then,

\begin{align*} \frac{3a}{a - 4} \amp = \frac{3b}{b-4}\\ 3a(b - 4) \amp = 3b(a - 4b)\\ 3ab - 12a \amp = 3ab - 12b\\ -12a \amp = -12b\\ a \amp = b \end{align*}

So \(f\) is an injection.

Use a proof by contradiction to show there is no \(a \in \R - \{ 4 \}\) such that \(f (a) = 3\text{.}\) Assume such an \(a\) exists. Then

\begin{align*} \frac{3a}{a - 4} \amp = 3\\ 3a \amp = 3a - 12\\ 0 \amp = -12 \end{align*}

and this is a contradiction. Therefore, for all \(x \in \R - \{ 4 \}\text{,}\) \(f (x) \ne 3\) and \(f\) is not a surjection.

(d)

\(g: ( \R - \{ 4 \}) \to ( \R - \{ 3 \} )\) defined by \(g(x) = \frac{3x}{x - 4}\text{,}\) for all \(x \in ( \R - \{ 4 \} )\text{.}\)

Solution.

The function \(g\) is a bijection. The proof that is an injection is similar to the proof that \(f\) is an injection in Task 7.3.2.c. To prove that it is a surjection let \(y \in \R - \{ 3 \}\text{.}\) Then, \(\frac{4y}{y - 3} \in \R - \{ 4 \}\) and

\begin{align*} g \left(\frac{4y}{y - 3} \right) \amp = \frac{3 \left(\frac{4y}{y - 3} \right)}{\left(\frac{4y}{y - 3} \right) - 4}\\ \amp = \frac{12y}{4y - 4(y - 3)}\\ \amp = \frac{12y}{12}\\ \amp = y\text{.} \end{align*}

This proves that \(g\) is a surjection.

3.

Let \(s\) be the function that associates with each natural number the sum of its distinct natural number divisors. This is called the sum of the divisors function. For example, the natural number divisors of 6 are 1, 2, 3, and 6, and so

\begin{align*} s(6) \amp = 1 + 2 + 3 + 6\\ \amp = 12 \end{align*}
(a)

Calculate \(s(k)\) for each natural number \(k\) from 1 through 15.

Solution.
\begin{align*} s(1) \amp = 1 \amp s(5) \amp = 6 \amp s(9) \amp = 13 \amp s(13) \amp = 14\\ s(2) \amp = 3 \amp s(6) \amp = 12 \amp s(10) \amp = 18 \amp s(14) \amp = 24\\ s(3) \amp = 4 \amp s(7) \amp = 8 \amp s(11) \amp = 12 \amp s(15) \amp = 24\\ s(4) \amp = 7 \amp s(8) \amp = 15 \amp s(12) \amp = 28 \amp s(16) \amp = 31 \end{align*}
(b)

Is the sum of the divisors function an injection? Is it a surjection? Justify your conclusions.

Solution.

The sum of the divisors function \(s\) is not an injection. For example, \(s(6) = s(11)\text{.}\) This function is also not a surjection. For example, for all \(x \in \N\text{,}\) \(s(x) \ne 2\) and for all \(x \in \N\text{,}\) \(s(x) \ne 5\text{.}\)

4.

Let \(\M_2 (\R)\) represent the set of all 2 by 2 matricies over \(\R.\)

(a)

Define \(\text{det: } \M_2 (\R) \to \R\) by

\begin{equation*} \text{det} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = ab - bc\text{.} \end{equation*}

This is the determinant function on the set of 2 by 2 matrices over the real numbers. Is the determinant function an injection? Is the determinant function a surjection? Justify your conclusions.

Solution.

The determinant function is not an injection. For example,

\begin{equation*} \text{det} \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} = \text{det} \begin{bmatrix} 1 \amp 2 \\ 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}

The determinant function is a surjection. To prove this, let \(a \in \R\text{.}\) Then

\begin{equation*} \text{det} \begin{bmatrix} a \amp 0 \\ 0 \amp 1 \end{bmatrix} = a\text{.} \end{equation*}
(b)

Define \(\text{tran: } \M_2 (\R) \to \M_2 (\R)\) by

\begin{equation*} \text{tran} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = A^T = \begin{bmatrix} a \amp c \\ b \amp d \end{bmatrix}\text{.} \end{equation*}

This is the transpose function on the set of 2 by 2 matrices over the real numbers. Is the transpose function an injection? Is the transpose function a surjection? Justify your conclusions.

Solution.

The transpose function is a bijection. To prove it is an injection let \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}, \begin{bmatrix} p \amp q \\ r \amp s \end{bmatrix} \in \M_2(\R)\) and assume that

\begin{equation*} \text{tran} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \text{tran} \begin{bmatrix} p \amp q \\ r \amp s \end{bmatrix}\text{.} \end{equation*}

Then \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \begin{bmatrix} p \amp q \\ r \amp s \end{bmatrix}\text{.}\) Therefore, \(a = p\text{,}\) \(b = q\text{,}\) \(c = r\text{,}\) and \(d = s\) and hence, \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \begin{bmatrix} p \amp q \\ r \amp s \end{bmatrix}\text{.}\) To prove that the transpose function is a surjection, let \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} \in \M_2(\R)\text{.}\) Then,

\begin{equation*} \text{tran} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\text{.} \end{equation*}
(c)

Define \(F: \M_2 (\R) \to \R\) by

\begin{equation*} F \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = a^2 + d^2 - b^2 - c^2 \end{equation*}

Is the function \(F\) an injection? Is the function \(F\) a surjection? Justify your conclusions.

Solution.

The function \(F\) is not an injection. For example

\begin{equation*} F \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} = 0 \qquad \text{and} \qquad F \begin{bmatrix} 1 \amp 1 \\ 1 \amp 1 \end{bmatrix} = 0\text{.} \end{equation*}

The function \(F\) is a surjection. To prove this, let \(y \in \R\text{.}\) Consider three cases.

  • If \(y = 0\text{,}\) then \(F \begin{bmatrix} 0 \amp 0 \\ 0 \amp 0 \end{bmatrix} = 0 = y\text{.}\)

  • If \(y \gt 0\text{,}\) then \(\sqrt{y} \in \R\) and \(F \begin{bmatrix} \sqrt{y} \amp 0 \\ 0 \amp 0 \end{bmatrix} = \left( \sqrt{y} \right)^2 = y\text{.}\)

  • If \(y \lt 0\text{,}\) then \(\sqrt{-y} \in \R\) and \(F \begin{bmatrix} 0 \amp \sqrt{-y} \\ 0 \amp 0 \end{bmatrix} = - \left( \sqrt{-y}^2 \right)^2 = y\text{.}\)
